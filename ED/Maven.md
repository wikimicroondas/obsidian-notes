# Maven
### Introducción
Primera parte de un ejercicio centrado en builders de Java.
### Descripción
En este bloque vamos a desarrollar un programa en Java que muestre un banner ASCII grande seguido de texto normal, y que todo el contenido se desplace verticalmente hacia arriba, simulando la introducción de una película.
