### Validación JSON: Schema
Conjunto de protocolos para asegurar identidades inequívocas a objetos y controlar el acceso de tipos de datos


"$schema"
"$id"
"title"
"description"
"properties"


urls: uri
números: minimum, maximum, multipleOf

## Asociación con documentos JSON
Podemos validar los datos mediante plantillas ya hechas `tipo.schema.json`.

```json
{
  "json.schemas": [
    {
      "fileMatch": ["datos/usuarios.json"],
      "url": "./schemas/usuario.schema.json"
    },
    {
      "fileMatch": ["datos/productos.json"],
      "url": "./schemas/producto.schema.json"
    },
    {
      "fileMatch": ["datos/pedidos.json"],
      "url": "./schemas/pedido.schema.json"
    }
  ]
}
```
